# Start here

This is the handoff package for the SQL Mate project. Everything you need to set up the repo and start Phase 1 with Claude Code is in this folder.

## If you are the human developer

Open `SETUP.md` and follow the steps in order. It walks you through installing prerequisites, initializing the git repo, pushing to GitHub, and starting your first Claude Code session. About 30 minutes start to finish.

## If you are Claude Code

Open `PHASE_1_KICKOFF.md`. It lists which docs to read, the decisions already settled, and Phase 1's done-when criteria. Do not start writing code until you have read everything it points to.

## What is in this folder

- `START_HERE.md` — this file
- `SETUP.md` — prerequisites and one-time setup for the developer
- `PHASE_1_KICKOFF.md` — the prompt and confirmed decisions for Claude Code
- `CLAUDE.md` — working norms and security non-negotiables (loaded automatically by Claude Code)
- `README.md` — internal-facing project README
- `.gitignore` — pre-configured for Tauri + Vite + Rust + secrets
- `docs/` — full architecture, security model, roadmap, and ADRs

Read in this order if you want full context: `README.md` → `docs/PROJECT_BRIEF.md` → `docs/ARCHITECTURE.md` → `docs/SECURITY_MODEL.md` → `docs/ROADMAP.md` → `CLAUDE.md`.

## Phase 0 status

Phase 0 (architecture and documentation) is complete. The next action is Phase 1 (the walking skeleton). All decisions needed to start Phase 1 are recorded in `PHASE_1_KICKOFF.md`.
